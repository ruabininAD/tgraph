В данной программе структура matrix имеет следующие методы:

     (m *Matrix) Identity()  - Делает матрицу единичной (вся из единиц)
     (m *Matrix) Set(row, col, value int) - устанавливает в матрицу смежности ориентированное ребро из вершины row в вершину col со значением веса value
     (m *Matrix) SetUnOrientedE(row, col, value int) - устанавливает в матрицу смежности неориентированное ребро между вершинами row и col со значением веса value
     (m *Matrix) Get(row, col int) int - получает значения из матрицы связности из вершины row в вершину col \
     (m *Matrix) Print() выводит матрицу смежности в консоль
     (m *Matrix) PrintLabel(text string) - выводит текстовый аргумент и с новой строки выводит матрицу смежности в консоль
     (m *Matrix) Render() - сохраняет матрицу в txt и вызывает программу py по визуализации графа
     (m *Matrix) Weigh() - заменяет значение весов всех ребер
