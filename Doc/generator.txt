В даннй программе есть слудующие методв генерации графов:


    1) func GetEndingGraph(vCount, eCount int) (*Matrix, error)
    Генерация на основе распределения Элдинга:
        Распределение Элдинга определяется параметрами
            k - параметр формы распределения
        	λ - параметр интенсивности распределения
        Используя это распределения получаем количество n входов для каждого узла.

        Генерируем n связей со случайными узлами.

    2)func GetOrientedAcyclicGraph(vCount, eCount int) (*Matrix, error)
    Генерация ориентированного ациклического графа

    3) func GetOrientedGraph(vCount, eCount int) (*Matrix, error)
    Генерация Ориентированного графа

    4) func GetUnorientedGraph(vCount, eCount int) (*Matrix, error)
    Генерация НеОриентированного графа

     5) NewMatrix(n int) *Matrix
     Генерация графа из n вершин без связей
